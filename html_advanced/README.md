# Holberton School - Web Front-End

## Description
This project focuses on the foundations of web development, specifically **HTML5**. The goal is to master the structure of web pages and understand the importance of **Semantic HTML** for accessibility and SEO.

## Learning Objectives
By the end of this project, you should be able to explain the following without external help:

### 1. HTML Guidelines
- Follow **W3C** standards: Use lowercase for tags and attributes.
- Always quote attribute values (e.g., `class="container"`).
- Ensure proper nesting of elements.

### 2. HTML5 Skeleton
- How to create the mandatory structure: `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>`.

### 3. Semantic HTML
Understanding the purpose of structural tags:
- `<header>`: Introductory content or navigational links.
- `<nav>`: Section for navigation links.
- `<main>`: The dominant content of the document.
- `<article>`: Self-contained, independent content.
- `<section>`: Generic standalone section of a document.
- `<aside>`: Content indirectly related to the main content (sidebars).
- `<footer>`: Footer for its nearest sectioning content.



### 4. Layout & Hierarchy
- **Div vs Span**: Using `<div>` for block-level containers and `<span>` for inline styling.
- **Headings**: Following the hierarchical order from `<h1>` to `<h6>` for document flow.
- **Lists**: Creating ordered (`<ol>`) and unordered (`<ul>`) lists.

### 5. Data & Media
- **Tables**: Structuring data using `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, and `<td>`.
- **Media Formats**:
    - **SVG**: Scalable vector graphics.
    - **PNG**: High quality with transparency.
    - **JPG**: Compressed photos.
    - **GIF**: Animated images.
- **Integration**: Embedding `<video>`, `<audio>`, and external content using `<iframe>`.

## Requirements
- All files must be **W3C compliant**.
- Use a text editor like **VS Code**, **Vim**, or **Emacs**.
- Every file must end with a new line.
